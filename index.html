Code_space is a personal site to highlight some of the code I have written with a particular focus on Physics simulations. As a Physics teacher I also aim to develop a series of tutorials on how to use the software in lessons as a student or teacher.

The languages I tend to code in are Java, Python, C# and Octave. As far as coding goes I am pretty much self-taught - Coursera, Udemy, Udacity and MITOpenCourseware have been fantastic sources of knowledge as well as the invaluable Stack Overflow site - and am certainly not an expert. I am certain that a lot of my code is bloated and inefficient, but it is slowly improving with time.

My main projects are the Physics Labs software that I am developing, currently including: OpticsLab and MechanicsLab. The links to an embedded version of the software, as well as tutorials, are to the left.

The "Physics Labs" website is currently in development and can be found here.

The physics_labs site is dedicated to the educational resources that I have created: Optics Lab, Mechanics Lab and soon Fields Lab. Although embedded versions and tutorials are still hosted here too, physics_lab will become the default repository for all things "labs" in the future.

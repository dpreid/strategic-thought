sudokuSolver

sudokuSolver does exactly what it says on the tin: solves sudoku. Nothing else. Although I think I will adapt it to also generate sudoku in the future. I'm calling it version 0.1 just now because I am sure that there are still a few problems to sort out: the output of information to the text area for one. I have tested it on many sudoku (from Easy to Evil) from the webdoku website, but I cannot guarantee that it solves all sudoku. It appears to solve them all so far.

The program was written in Java using the WindowsBuilder plugin on Eclipse for the GUI.
